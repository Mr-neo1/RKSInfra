# SEO Implementation Guide

This document outlines the SEO optimizations implemented for the RKS Infra website.

## Logo Setup

**Important:** The logo needs to be manually copied to the public folder:

1. Copy `logo/RKSInfra-logo.png` to `public/logo.png`
2. The Header and Footer components are already configured to use `/logo.png`
3. The logo is optimized using Next.js Image component for better performance

## Google Search Console Setup

To get listed in Google and other search engines:

1. **Submit Sitemap**
   - Go to Google Search Console
   - Submit: `https://rksinfra.com/sitemap.xml`

2. **Verify Ownership**
   - Add verification meta tag to `app/layout.tsx` in the verification section
   - Or use HTML file upload method

3. **Bing Webmaster Tools**
   - Submit sitemap to Bing
   - Verify ownership similarly

4. **Other Search Engines**
   - Yandex Webmaster
   - Baidu Webmaster (for China)
